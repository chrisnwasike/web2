# Chris Nwasike - Business Systems & Websites for Nigerian SMEs

Senior web developer portfolio focused on operational websites, SME business systems, and conversion-driven web applications. Built to position Chris as *the guy who understands SME business systems online*.

## 🔧 Tech Stack
- HTML5 / CSS3 (custom, no framework)
- JavaScript (Vanilla, GSAP for scroll animations)
- PHP / MySQL (backend references in projects)
- Responsive, mobile-first architecture

## 🚀 Local Development
Simply open `index.html` in any modern browser. No build step required.

```bash
# Clone the repo
git clone https://github.com/chrisnwasike/web.git

# Open locally
open index.html   # macOS
start index.html  # Windows
```
